SCREENSHOT NAMING CONVENTION
=============================

Add your screenshots to this folder with the following naming convention:

step-1.png   - Welcome screen
step-2.png   - Navigation tabs
step-3.png   - Financial Summary cards
step-4.png   - Trends & Visualizations charts
step-5.png   - Financial Action Plans
step-6.png   - Reports home page
step-7.png   - Available Reports selection
step-8.png   - Transaction table
step-9.png   - Smart Learning / Repeating Transactions
step-10.png  - Financial Consultation (Finn)
step-11.png  - GMS Overview dashboard
step-12.png  - GMS Action Plan
step-13.png  - GMS Health Check intro
step-14.png  - Capitation Income card
step-15.png  - Practice Support card
step-16.png  - Leave Payments card
step-17.png  - Chronic Disease Management card
step-18.png  - Cervical Screening card
step-19.png  - STC Payments card
step-20.png  - Admin Settings (Upload Data)
step-21.png  - Meet Cara button
step-22.png  - Tour completion screen

TIPS:
- Use PNG format for best quality
- Recommended dimensions: 1280x720 or similar 16:9 ratio
- Screenshots should show the highlighted element from the tour
- You can use Windows Snipping Tool (Win+Shift+S) or similar
